Warenkorb Menü hinzufügen



In dieser Challenge lernst du den einfachsten Weg, um Waren zu einem Warenkorb hinzuzufügen. Hierfür erstellen wir 3 verschiedene Container für den Namen, den Preis und die Menge der Ware. Diese Container werden nun mithilfe unseres Codes befüllt.

1. Erstelle ein Array namens menus.

Array menus wird die Essennamen von Menüs in unseren Warenkorb enthalten.

Tipp: Bitte definiere Variablen immer ganz oben in deinem Script. Die Definitionsreihenfolge ist hier egal.
2. Erstelle ein Array namens prices.

Array prices wird die Prise von Menüs in unseren Warenkorb enthalten.

Tipp: Bitte definiere Variablen immer ganz oben in deinem Script. Die Definitionsreihenfolge ist hier egal.
3. Erstelle ein Array namens amounts.

Array amounts wird die Menge von Menüs in unseren Warenkorb enthalten.

Tipp: Bitte definiere Variablen immer ganz oben in deinem Script. Die Definitionsreihenfolge ist hier egal.
4. Füge den Arrays Werte hinzu.

Füge dem Array menus Essennamen hinzu. Die Namen sollen vom Datentyp string sein und dürfen keine leeren Zeichen am Anfang und Schluss des Names enthalten. Z.B.: "Pizza Salami" und nicht " Pizza Salami ".

Füge dem Array prices Nummern hinzu. Die Nummern sollen vom Datentyp number sein und dürfen Kommazahlen sein. Z.B.: 1.99 oder 2.0 und nicht "1.99".

Füge dem Array amounts ganze Zahlen hinzu. Die Zahlen sollen vom Datentyp number sein (Keine Kommazahlen!). Z.B.: 1 oder 10 und nicht "1" oder 1.5.

Tipp: Achte darauf, dass alle drei Arrays die gleiche Länge haben.
5. Erstelle eine Hilfsfunktion namens getValueFromInput

Erstelle eine Hilfsfunktion namens getValueFromInput, welche dir den Wert von einem beliebigen Input Feld zurückgibt. Die Funktion getValueFromInput sollte einen Parameter haben, welcher der ID von einem Input Feld entspricht.

6. Erstelle eine Hilfsfunktion namens getMenuFromInput

Erstelle eine Hilfsfunktion namens getMenuFromInput, welche den Wert von dem Input Feld mit der ID menu validiert und zurückgibt. Dem Wert sollten alle leeren Zeichen am Anfang und am Ende entfernt werden.

Tipp:
  • Nutze die Hilfsfunktion getValueFromInput, um die Werte von den Input-Feldern auszulesen.
  • Versuche auch das Internet für die Lösung des Problems zu verwenden.
  • Nutze Stichworte beim Suchen, um genau dein Problem zu beschreiben.
  • Beispiel: "Javascript, String, Remove empty spaces from both sides"
7. Erstelle eine Hilfsfunktion namens getPriceFromInput

Erstelle eine Hilfsfunktion namens getPriceFromInput, welche den Wert von dem Input Feld mit der ID price validiert und zurückgibt. Der Wert sollte vom Datentyp number sein.

Tipp:
  • Nutze die Hilfsfunktion getValueFromInput, um die Werte des Input Felds herauszulesen.
  • Versuche im Internet Lösungen zu finden für deine Probleme.
  • Nutze Stichworte, um genau dein Problem zu beschreiben.
  • Beispiel: "javascript string convert to number"
8. Erstelle eine Funktion für den Button Menü hinzufügen

Erstelle eine Funktion, welche vom Button Menü hinzufügen per Klick aufgerufen wird.

Tipp: Untersuche den Button mit den Developer Tools vom Browser, um den Funktionsnamen herauszufinden der dort bereits existiert.
9. Erweitere die Funktion onAddMenu

Erweitere die Funktion onAddMenu, sodass der Inhalt der Input-Felder am Ende der Arrays menus und prices hinzugefügt werden. Auch am Ende des Arrays amounts sollte eine 1 hinzugefügt werden.

Tipp: Nutze die Hilfsfunktionen getMenuFromInput und getPriceFromInput, um die Werte von Input-Feldern herauszulesen.
10. Erstelle eine Hilfsfunktion namens getMenuIndex

Erstelle eine Hilfsfunktion namens getMenuIndex, welche einen Parameter namens menu hat. Das Ergebnis der Funktion ist der Index, wo sich der Parameter menu in dem Array menus befindet. Gebe -1 zurück, wenn das Ergebnis nicht gefunden werden kann.

Tipp:
  • Versuche im Internet Lösungen zu finden für deine Probleme.
  • Nutze Stichworte beim Suchen, um genau dein Problem zu beschreiben.
  • Beispiel: "Javascript index of element in array"
11. Ändere die Funktion onAddMenu

Ändere die Funktion onAddMenu so, dass die Werte nur hizugefügt werden, wenn der Wert des Input Felds mit der ID menu sich noch nicht in dem Array menus befindet. Andernfalls, wenn der Wert sich in Array menus befindet, dann finde die Index-Stelle und erhöhe den Wert in Array amounts in dieser Stelle um 1.

Tipp:
    Nutze die Hilfsfunktion getMenuIndex, um zu überprüfen, ob das Menü sich bereits in dem Array menus befindet.
